
Movie Recommender App using Streamlit
=====================================

This is a simple movie recommender GUI built with Python and Streamlit.

📝 Files Included:
------------------
- app.py         → The main Streamlit app
- dataset.csv    → Sample dataset used for recommendations

🚀 How to Run:
--------------
1. Make sure you have Python 3 installed.

2. Install required package:
   pip install streamlit pandas

3. Run the app using:
   streamlit run app.py

4. A browser tab will open automatically. If not, open the URL shown in the terminal.

📁 Output:
----------
- A web-based app where you can choose Gender and Genre preferences to get movie recommendations.

Have fun exploring movie suggestions!
